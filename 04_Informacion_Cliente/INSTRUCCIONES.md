# Información del Cliente

Subir aquí cualquier archivo adicional para complementar la propuesta:

- Observaciones o correcciones a los documentos técnicos
- Catálogos de proveedores o cotizaciones
- Información técnica complementaria
- Cualquier otro documento relevante

**Para subir**: Clic en "Add file" → "Upload files" → Arrastrar archivos → "Commit changes"

**Para reportar observaciones**: Usar la pestaña "Issues" del repositorio.
